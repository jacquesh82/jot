jot serve

Starts the server. Open http://localhost:3000 to register.

🔑

No account needed

Identity is a local UUID with a friendly name. Link extra devices via QR code. No email, no password, no tracking.

📦

One binary

CLI, API server, and web app compiled into a single statically linked executable. Zero runtime dependencies.

🔒

Encrypted exports

Export all your notes as JSON, optionally locked with AES-256-GCM + PBKDF2. End-to-end encryption in progress.

🌍

Runs anywhere

Static musl binaries for Linux and Windows. Native builds for macOS. Self-host in seconds, no Docker required.

🗂️

Boards & sharing

Organise notes into boards. Share boards or individual notes with other identities, with per-user access control.

⚡

Unix-friendly CLI

Pipe text straight into jot. Open your $EDITOR. Query from scripts. Feels like a tool, not an app.

Get started in seconds

$ jot serve
Server running at http://localhost:3000
Database schema: up to date (v4)
$ echo "buy milk" | jot add
$ jot list
buy milk · default · just now
$ jot tui
Launches the interactive TUI
